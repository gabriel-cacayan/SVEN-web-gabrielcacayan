# Installation

## Requirements

-   PHP version must be 8.0 and higher

## Clone

-   git clone https://github.com/gabriel-cacayan/SVEN-web-gabrielcacayan.git
    -cd SVEN-web-gabrielcacayan
-   composer install
-   create an env and a database
-   php artisan migrate
-   php artisan serve
